Для кого нужна семантика?

Для необычных устройств

Сейчас сайты можно открывать на различных смарт часах и других похожих устройствах. На них не будет отображаться графический интерфейс, но они умеют выделять контент, как режим чтения в обычном браузере.

Для людей с проблемами со зрением

Для незрячих или частично незрячих всё сложнее. Основной инструмент для просмотра сайтов не браузер, который отрисовывает страницу, а скринридер, который читает текст со страницы вслух.

Для режима чтения

Режим чтения — это режим просмотра веб-страницы, который обеспечивает отображение на экране только текста и изображений к нему без лишних элементов веб-страницы, что позволяет пользователю сосредоточиться на чтении текстового контента.

Семантическая вёрстка
— подход к разметке, который опирается не на содержание сайта, а на смысловое предназначение каждого блока и логическую структуру документа. Даже в этой статье есть заголовки разных уровней — это помогает читателю выстроить в голове структуру документа. Так и на странице сайта — только читатели будут немного другими.
Screenreader
— это один из видов вспомогательных технологий, предназначенных для слепых людей, имеющих плохое зрение. C помощью такого приложения пользователь может постигать визуальную информацию без использования зрения, например, озвучивания, звуковых значков или Брайлевского дисплея.
